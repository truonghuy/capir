File format: <numVirtualStates> <numAgentProp> <numHelperProp> <numMonsterProp> <numSpecLocProp>. In the current implementation, numAgentProb = numHelperProp = 3, numMonsterProp = 4 or 5 depending on whether there is any hit point.

int [] CapirBox.existingNPCTypes [private]

This array stores the index of the Q function for each NPC type.

Some map could contain less NPC types than normally. existingNPCTypes[i] = -1 when there is no such NPC type in the map. existingNPCTypes[i] = j >=0 when QValues[j] is the Q-function of NPCType i.

List<double[,,]> CapirBox.QValues

Stores the Q values of the NPC types existing in the map.

This matrix is used to access Qvalues of an NPC given its index. Indices of QValues are QValues[existingNPCTypes[npcTypes[i]]][state, agentAction, helperAction].
